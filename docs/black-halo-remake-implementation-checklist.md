# Black Halo Remake Implementation Checklist

## 현재 기준

- 이 체크리스트는 리메이크의 실제 구현 진행표다.
- 현재 구현의 기준 런타임은 `web` 플레이어블 빌드다.
- `Godot` 프로덕션 이식은 이 문서의 다음 마일스톤으로 이어진다.

## 진행도 스냅샷

- 전체 리메이크 프로그램: `33%`
- 마일스톤 1 `웹 리메이크 기초`: `100%`
- 마일스톤 2 `월드 재구성/콘텐츠 확장`: `40%`
- 마일스톤 3 `Godot 프로덕션 마이그레이션`: `0%`
- 마일스톤 4 `연출/밸런스/폴리시`: `0%`
